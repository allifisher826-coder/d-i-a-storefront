
## Structure

```
index.html              → Main storefront (single page)
order-confirmation.html → Post-order confirmation
404.html                → Branded 404 page
store.html              → Redirects to /#shop
public/mockups/         → Product photography
data/products.json      → Product catalog data
config/                 → API keys (.env, gitignored)
scripts/                → Printful sync, product generation tools
```
