┌─────────────────────────────────────────────────────────────────────────────┐
│ 🎯 HOW TO USE (3 SIMPLE STEPS)                                              │
└─────────────────────────────────────────────────────────────────────────────┘

  1️⃣  PUT YOUR MOCKUP IN THE FOLDER

      Create two files in mockups-input/:
      
      mockups-input/DIA-TEE-001.jpg        (your product photo/mockup)
      mockups-input/DIA-TEE-001.txt        (product details)

  2️⃣  RUN THE CONVERSION COMMAND

      npm run pod:from-mockup -- --sku DIA-TEE-001

  3️⃣  DONE. PRODUCT IS LIVE.

      Visit: doitanyway.netlify.app
      See your product on the store

┌─────────────────────────────────────────────────────────────────────────────┐
│ 🔄 WHAT HAPPENS WHEN YOU RUN THE COMMAND                                   │
└─────────────────────────────────────────────────────────────────────────────┘

  npm run pod:from-mockup -- --sku DIA-TEE-001
  
    ↓ AI analyzes your mockup image
    ├─ Identifies design placement
    ├─ Detects colors and style
    └─ Extracts design specifications
    
    ↓ System generates:
    ├─ Embroidery-ready design specs
    ├─ Product copy (SEO + conversion)
    └─ Product database entry
    
    ↓ Publishes to fulfillment
    ├─ Syncs to Printful
    ├─ Creates variants
    └─ Sets up production
    
    ↓ Updates your store
    ├─ Adds product card
    ├─ Pushes to GitHub
    └─ Deploys to Netlify
    
    ↓ 
    🎉 LIVE ON STORE

┌─────────────────────────────────────────────────────────────────────────────┐
│ 💰 BATCH UPLOAD (Multiple Products)                                         │
└─────────────────────────────────────────────────────────────────────────────┘

  1. Add all mockups to mockups-input/
     
     mockups-input/
     ├── DIA-TEE-001.jpg + .txt
     ├── DIA-HOODIE-001.jpg + .txt
     ├── DIA-TRACKSUIT-001.jpg + .txt
     └── DIA-SHORT-001.jpg + .txt

  2. Run batch command

     npm run pod:from-mockups:all

  3. All products process in parallel

     ✓ Generate design specs
     ✓ Generate copy
     ✓ Sync to Printful
     ✓ Publish to store

  4. All go live together

     ~10 minutes for 5 products
